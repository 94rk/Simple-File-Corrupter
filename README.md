# FileFryer

FileFryer is a lightweight console utility written in C# that lets you corrupt files and change their size.

## Features

- Corrupt any file with random bytes
- Change a file's size
- Clean, simple command-line interface
- Automatic file size detection
- Looping workflow for multiple files

## How to Use

1. Run the tool
2. Drag & drop or type a file path
3. Choose:
  1 → Corrupt
  2 → Corrupt + Resize

FileFryer does the rest.

## ⚠️ Disclaimer ⚠️

This tool WILL permanently damage files. Always test on copies, FileFryer takes no responsibility for any loss.
